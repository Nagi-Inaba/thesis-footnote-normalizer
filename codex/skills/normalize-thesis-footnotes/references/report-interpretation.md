# Report interpretation

- `first`: first matched occurrence of a registered source in document order.
- `repeat`: later matched occurrence that is not an immediate single-source repeat.
- `ibid_candidate`: current and previous footnotes each match exactly the same one source.
- `unmatched`: no registered alias was found. The note may be explanatory or the registry may be incomplete.
- `multiple_match`: more than one registered source was found. This may be a valid multi-source note.
- `unused_bibliography`: a registry record was not matched. Do not delete it automatically.

Every classification is a screening result. It is not proof that the existing or proposed citation text is correct.
